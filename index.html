Meus projeos

O Scratch, uma plataforma de programação visual desenvolvida pelo MIT, permite que pessoas de todas as idades criem seus próprios jogos e animações. Um ótimo exemplo do potencial criativo do Scratch é um jogo de cartas digital, que combina lógica, design e interatividade em um ambiente acessível e educativo.

projeto Recomendador de Filmes desenvolvido com JavaScript é uma aplicação simples, interativa e útil que tem como objetivo sugerir filmes ao usuário com base em suas preferências. Esse tipo de sistema é uma introdução prática ao mundo do desenvolvimento web e aos conceitos básicos de lógica de programação, manipulação de dados e interação com o usuário.

projeto Linhas Feitas de Círculos Consecutivos em JavaScript é uma demonstração criativa de como usar a programação para gerar formas geométricas e padrões dinâmicos na tela. A ideia principal desse projeto é criar uma linha composta por círculos desenhados sequencialmente, de forma que cada círculo seja posicionado de maneira a formar uma linha contínua ou até padrões mais complexos.
